# HR-Attrition-Analysis-Dashboard
End-to-End HR -Attrition-Analysis using Microsoft Excel, and Power BI to identify employee attrition patterns and key factors affecting employee turnover. 

## Project Overview
This project analyzes employee attrition patterns to understand why employees leave the organization and identify key factors influencing attrition.

## Tools Used
- Excel (Data Cleaning)
- Power BI (Data Visualization & Dashboard)

## Key Insights
- Attrition rate by department
- Attrition by age group
- Attrition by job role
- Impact of salary and years at company
- Gender wise attrition analysis

## Dashboard Features
- Interactive Power BI dashboard
- Employee demographics analysis
- Attrition trend visualization
